It's all about cards, colors, and chance.


As with most games, the goal of Coloretta is to get the most points. Points are obtained by collecting as many cards of the same color as possible however too much variety in colors will cost you points. (I'll explain more about it in scoring)

Round Setup

Each round starts with each player getting a different color dealt to them. In addition, if there are three players there will be three game stacks setup, four stacks for 4 players, etc. 
The deck is randomized and the "Last Cycle" card is placed 15 cards from the bottom. The deck consists of 63 color cards (7 different colors and 9 cards of each), 10 '+2' cards, and 3 wilds.

NOTE: A 3 player game uses 6 colors rather than 7.
Game Play
The first player is determined randomly. A player has two options on their turn. Either drawing from the deck and placing the card on a game stack *or* picking a game stack.
Draw from the deck and placing a card
A player must draw from the deck if all of the game stacks are empty.
A player may not place the card on a stack that a player has picked up. (Once everyone has picked up a stack then the cycle is complete and all stacks are 'empty' again.)
A game stack is full once it has 3 cards in it. A card cannot be placed on a full stack.
A player cannot draw from the deck if all un-picked game stacks are full.
Pick a game stack
Instead of drawing a card the player may choose to pick one of the game stacks. Picking a game stack places the cards in the selected stack into the player's hand.
A player cannot pick an empty stack.
After picking a stack the player does not get another turn until all players have picked a stack. After all players have selected a stack a new 'cycle' starts.
Once a stack has been picked it is deactivated until the next cycle. In each 'cycle' each stack will be selected by different players which is why there are as many stacks as there are players.
Scoring
Scoring is not final until the round is over. The round is over when the cycle in which the "Last Cycle" card is drawn is finished. At this time the player's hand score is added to the player's previous round scores. The player's hand is cleared and another round is setup. There are 4 rounds in a game by default.
Scoring is based on Fibonacci numbers and how many cards of in set you have. (Cards are grouped into a sets based on color.)  The three largest sets are added up and any remaining sets are a subtracted as a penalty. It is to your advantage to select game stacks that will make your hand contain no more than three different colors although you'll probably always have more than three colors. NOTE: As you can see in the scoring chart below. Getting more than 6 cards in a set does not give you more points.
If the player has any wild cards then they are grouped in color sets that will give the player the most points.  A wild card will never be grouped with a color set that would subtract points.
+2 cards add 2 points to the player's hand.
Although the player would have to have horrible luck, it is possible to get a negative score.

Keyboard Shortcuts
Once a game is started you will not need to use a mouse until the game is over. Below is a table that maps game play commands to keyboard characters.
 
Keyboard Key Action
d Draw from the deck
1 Place drawn card on game stack 1.
OR
Place game stack 1 in your hand if you have not drawn a card yet.
2 Place drawn card on game stack 2.
OR
Place game stack 2 in your hand if you have not drawn a card yet.
3 Place drawn card on game stack 3.
OR
Place game stack 3 in your hand if you have not drawn a card yet.
4 Place drawn card on game stack 4.
OR
Place game stack 4 in your hand if you have not drawn a card yet.
5 Place drawn card on game stack 5.
OR
Place game stack 5 in your hand if you have not drawn a card yet.
